# RWE-Plus
Maybe better version of official rain world editor

uses [drizzle](https://github.com/PJB3005/Drizzle) ro render levels

![image](https://github.com/timofey260/RWE-Plus/blob/master/files/image.png?raw=true)


## Available Editors
* Geometry editor
* Tile editor
* Light editor
* Level resizing
